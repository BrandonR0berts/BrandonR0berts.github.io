Main Concepts Developed

The Ghosts within this Pacman recreation are fully AI controlled, using a stack based finite state machine. The player can also control the Ghosts if they select them from the main menu. By default the player will start the level controlling pacman, and will have to collect all of the dots before dying to the ghosts that are chasing them. If the player selects to play as a ghost then pacman will become fully AI controlled, using the same stack based AI system as the ghosts use.
