Super Mario Bros 3

Developed using SDL2 in C++ for a first year game development task.

As a submission for my first semester game development module we were tasked with following tutorials on creating the one screen arcade Mario Bros game. As an extension to this task we were invited to expand on the task of we wished, so long as it still fit the marking criteria. Therefore, I decided to use some prior experience programming and aim at re-creating as much of the first world of the SNES Mario Bros 3 game as I could in the time frame. Overall, the visuals and controls for the first world, and the first level were able to be created in the could of months time frame, as is shown in the video link below. The main difficulty within this project came from a combination of it being one of my first large scale solo projects undertaken, and the sheer amount of objects, items and systems requred to be put in place to accomidate everything.
